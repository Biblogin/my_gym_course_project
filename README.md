# Прогнозирование посещаемости тренажерного зала

## Описание проекта
Учебный проект посвящен разработке и исследованию моделей машинного обучения для предсказания точного количества людей в тренажерном зале в определенный момент времени на основе дня недели, времени суток, температуры на улице и начала семестра. Данный проект, мог бы лечь в основу реального приложения для фитнес-клуба, показывающего оптимальное время для тренировок без очередей.

## Цели проекта
1. Сравнить эффективность различных методов регрессии.
2. Выявить наиболее точную модель для данного датасета.
3. Визуализировать предсказания моделей для понимания их работы.

## Используемые модели
Для решения задачи были реализованы и обучены следующие методы:
* **Линейная регрессия (OLS)** — классический аналитический метод.
* **Линейная регрессия (SGD)** — итеративный метод градиентного спуска.
* **Ядерная регрессия (RBF)** — метод, позволяющий улавливать нелинейные зависимости с помощью радиально-базисного ядра.
* **Случайный лес (Random Forest)** — ансамблевый метод, разбивающий данные на логические интервалы (кусочно-постоянная аппроксимация).

## Для запуска проекта на Unix-подобной системе или WSL (Windows SubSystem for Linux) выполните следующие команды:

### 1. Для установки необходимых библиотек выполните следующее:
Перейдите в директорию проекта и выполните команды:
(для установки на Debian/Ubuntu системах, на других используйте свой пакетный менеджер)

```bash
    sudo apt install python3.12-venv -y
    python3 -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt
```
    
### 2. После чего запустить main.py:

```bash
    python main.py
```
